## Run Instructions

### 1. Start the Server (Development)
Runs with `nodemon` for auto-restarts:
```bash
npm run dev
```

### 2. Start the Server (Production)
Runs with standard `node`:
```bash
npm start
```

### 3. Run MongoDB Script
Tests the database connection and adds a note:
```bash
node mongo.js <password>
```
